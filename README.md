

# This is to demonstrate how the blockchain works with a web application. 


## Usage

  - npm install
  - npm start (or "npm devstart" to start as deamon)
  - Launch "http://localhost:3000"

## Developing
  - Express Framework
  - EJS Template Engine
  - MongoDB (hosted on mLab)
